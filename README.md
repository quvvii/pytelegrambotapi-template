# PyTelegramBotAPI Template Bot

This project was created in order to help beginners get used to PyTelegramBotAPI library.


## Installation

Create `.env` file:
```env
TOKEN=YOUR_BOT_TOKEN
ADMIN_ID=YOUR_TELEGRAM_ID
```
Running on Windows:
`python bot.py`

Linux / MacOS:
`python3 bot.py`
